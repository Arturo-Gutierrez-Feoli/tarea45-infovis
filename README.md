## Descripción

Visualizaciones interactivas de ventas globales de videojuegos implementadas en formato SVG con la especificación ODViz (Open Data Visualization). Los datos están embebidos directamente en cada archivo SVG, permitiendo interactividad mediante el enlace bidireccional entre elementos visuales y filas de datos.

## Archivos incluidos

| Archivo | Tipo | Descripción |
|---|---|---|
| `index.html` | Página principal | Enlaza todas las visualizaciones con análisis |
| `grafica1_barras.svg` | Unidimensional | Ventas por género — gráfica de barras |
| `grafica2_pastel.svg` | Unidimensional | Participación de mercado por plataforma — pastel |
| `grafica3_histograma.svg` | Unidimensional | Ventas por año de lanzamiento — histograma |
| `grafica4_dispersion.svg` | Bidimensional | Ventas NA vs Europa por género — dispersión |
| `grafica5_barras_agrupadas.svg` | Bidimensional | Ventas por género y región — barras agrupadas |
| `grafica6_burbujas.svg` | Multidimensional | Análisis de 5 variables simultáneas — burbujas |

## Fuente de datos

VGChartz — Base de datos global de ventas de videojuegos (2000–2013)

## Formato utilizado
